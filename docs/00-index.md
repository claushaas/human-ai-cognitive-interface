# Human–AI Cognitive Interface — Índice da Documentação (pt-BR)

Esta pasta contém uma documentação **ampla, coerente e altamente estruturada** derivada do diretório `info/`, que é a **Fonte de Verdade primária** deste repositório.

Princípios operacionais desta documentação (conforme a Fonte de Verdade):

- Não introduz novas features/etapas fora do material existente.
- Reorganiza, consolida e reescreve para clareza/estrutura, preservando rastreabilidade conceitual.
- Inconsistências entre fontes foram resolvidas por uma regra única de precedência (ver `12-constitution.md`) e registradas como decisões canônicas (ver `09-open-issues-and-gaps.md`).
