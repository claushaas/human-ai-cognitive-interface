# app/test/

Diretório reservado para helpers compartilhados de teste dentro do app.

Nesta fase, ainda não há engine, contratos ou dados canônicos implementados em código. Futuramente, helpers de teste como factories de domínio, fixtures TypeScript e utilitários de mock viverão aqui.

Os testes de unidade e integração residem em `tests/unit/` e `tests/integration/`. Fixtures JSON e golden tests residem em `tests/fixtures/`.
